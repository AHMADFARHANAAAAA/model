================================================================================
🧠 MODEL OVERVIEW
================================================================================

ARCHITECTURE: LSTM-based Deep Knowledge Tracing

INPUT:
  Student interactions: [(q1,correct), (q2,incorrect), ...]
  
LAYERS:
  1. Question Embedding (50 dims)
  2. Correctness Embedding (50 dims)
  3. LSTM Layer 1 (128 units)
  4. LSTM Layer 2 (128 units)
  5. Dense Layer (64 units, ReLU)
  6. Output Layer (1 unit, Sigmoid)
  
OUTPUT:
  Probability of answering next question correctly (0-1)
  
PARAMETERS:
  Total: ~524,000
  Trainable: ~524,000
  Size on disk: ~50-100 MB

TRAINING:
  Loss: Binary Cross-Entropy
  Optimizer: Adam (lr=0.001)
  Regularization: Dropout (0.3), L2 decay (1e-5)
  Early stopping: Patience=10 epochs
